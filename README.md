Rock-Paper-Scissors Game (Python)

A simple Python game where you play Rock, Paper, Scissors against the computer.

---

📸 Features

🎲 Random computer choice
😂 Funny win/lose messages
📊 Score tracking system
🔁 Unlimited gameplay
💻 Simple command-line interface

---

📋Requirements
🐍Python 3.x

---

▶️How to Run

In bash write: python game.py

---

🎮How to Play
Type one of: rock, paper, or scissors
Computer randomly picks its move
Winning rules:
Rock beats Scissors
Scissors beats Paper
Paper beats Rock
Score updates after each round
Type yeah to continue or anything else to quit

---

Example
Choose rock, paper, or scissors: rock
Computer chose: paper
Huh such a loser

Score -> You: 0 | Computer: 1

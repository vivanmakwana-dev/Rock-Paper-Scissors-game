![Game](https://img.shields.io/badge/Type-Console%20Game-orange)

A simple Python game where you play Rock, Paper, Scissors against the computer.

---

📋 Requirements

* 🐍 Python 3.x installed
* 💻 A terminal or command prompt

---

🧩 How it works

* 🤖 The computer randomly selects rock, paper, or scissors
* 🎮 You choose your move
* 🏆 The winner is decided using standard Rock-Paper-Scissors rules
* 📊 Score is tracked for both player and computer
* 🔁 You can play unlimited rounds

---

🎮 How to Play

* ✊ Type `rock`, `paper`, or `scissors`
* 🤖 The computer will randomly choose its move
* 🏆 Win the round by choosing the stronger option:

  * ✊ Rock beats ✂️ Scissors
  * ✂️ Scissors beats 📄 Paper
  * 📄 Paper beats ✊ Rock
* 📊 Scores are updated after each round
* 🔁 Type `yeah` to play again or anything else to quit

---

▶️ How to Run

1. Install Python (if not installed)
2. Download or clone this repository
3. Open a terminal in the project folder
4. Run:

```bash
python game.py
```
